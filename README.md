# Roguelighter Engine

Roguelighter is a free, open-source, easy-to-use game engine
for 2D game development.

### Development

Clone the repo
`git clone https://github.com/roguelighterengine/roguelighter.git`

Install the dependencies
`npm install`

##### Desktop app development

`cd apps/app`
`npm install`
`npm run dev`

##### Website development

`cd apps/website`
`npm install`
`npm run dev`

##### Library development

Before development, run `npm run typegen` on `apps/app` for type generation
head over to `packages/roguelighter-core` and have fun!
